# Sistema-para-uma-bicicletaria-em-C

Trabalho final para a conclusão da matéria de Introdução à Programação de Computadores, pertencente ao primeiro período do cursos de Sistemas de Informação - UFU.

O objetivo era desenvolver um simples sistema de informação para uma bicicletaria. O sistema deveria seguir os seguintes critérios:

* For codificado em C;
* Tiver a função main( ) do template sem alteração;
* A declaração de variáveis e estruturas não poderá ser global;
* A função menu( ) tiver, no mínimo, 7 opções, além da opção sair, e, todas, com
funcionamento correto.

Além disso, era necessário ter pelo menos um uso de vetor, struct, ponteiro e função, além de utilizar no mínimo algumas funções para escrita e leitura em arquivos.

##

A apresentação do código contava com duas modificações propostas pelo professor, os grupos tinham 10 minutos para resolver cada uma das mudanças propostas. Em caso de sucesso haveria um acréscimo na nota, caso contrário os integrantes teriam uma segunda e última chance para a realização das alterações.

As mudanças propostas no sistema foram as seguintes:

1. Fazer uma função que mostre todas as bicicletas que estão alugadas e a soma total do valor dessas bicicletas
2. Fazer uma função que permitisse reajustar o preço dos produtos todos de uma vez

Dessa forma, o repositório apresenta duas versões de código, uma com a versão original do sistema e a outra com as mudanças exigidas pelo professor durante a apresentação.
